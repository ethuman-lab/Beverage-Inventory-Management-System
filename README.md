# Beverage Inventory Management System

A multi-location beverage inventory tracking system built in Microsoft Excel to solve a real operational problem across four bar locations.

## The Problem
Manual inventory counts across four locations with different product mixes were inconsistent, time consuming, and generating costly ordering errors.

## The Solution
Designed and built a structured Excel workbook with location-specific inventory sheets, automated per-unit cost calculations from case and keg pricing, and a summary dashboard tracking total inventory value by location.

## Results
- Eliminated 80 labor hours per week
- Generated $22,000+ in annual cost savings
- Improved inventory accuracy across all four locations

## Tools Used
Microsoft Excel, Advanced Formulas, Data Modeling
